Học lập trình Shader từ cơ bản đến nâng cao bằng tiếng Việt. Lý thuyết rõ ràng, code ví dụ thực tế, bài tập trắc nghiệm ngay trên trình duyệt.

Bắt đầu học ngay Xem giáo trình
MyFirstShader.shader 
Shader "Custom/MyFirstShader" {
  Properties {
    _Color ("Main Color", Color) =
      (1,1,1,1)
    _MainTex ("Texture", 2D) =
      "white" {}
  }
  SubShader {
    Pass {
      CGPROGRAM
      // vertex & fragment shaders
      #pragma vertex   vert
      #pragma fragment frag
      ENDCG
    }
  }
}
